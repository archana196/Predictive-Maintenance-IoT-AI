# Data Dictionary

## AI4I 2020 Predictive Maintenance Dataset

### Dataset Overview

The AI4I 2020 Predictive Maintenance Dataset is designed for predictive maintenance applications in manufacturing environments. It contains machine operating parameters, product quality information, and failure indicators used to predict whether a machine will fail during operation.

*Total Records:* 10,000  
*Total Features:* 14  
*Target Variable:* Machine Failure

---

## Feature Categories

### 1. Identification Features

These columns uniquely identify records and products.

- UDI
- Product ID
- Type

### 2. Sensor Features

Machine operating measurements collected from sensors.

- Air Temperature [K]
- Process Temperature [K]
- Rotational Speed [rpm]
- Torque [Nm]
- Tool Wear [min]

### 3. Target Variable

The primary variable used for predictive maintenance modeling.

- Machine Failure

### 4. Failure Type Indicators

Specific machine failure categories.

- TWF (Tool Wear Failure)
- HDF (Heat Dissipation Failure)
- PWF (Power Failure)
- OSF (Overstrain Failure)
- RNF (Random Failure)
