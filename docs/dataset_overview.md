# AI4I Predictive Maintenance Dataset Overview

## Project

Contextual Predictive Maintenance (IoT Edge AI)

## Dataset Description

The AI4I Predictive Maintenance Dataset contains industrial machine sensor data used to predict machine failures. The dataset simulates real-world manufacturing environments and includes operational parameters, machine conditions, and failure information.

## Dataset Objectives

* Understand machine behavior through sensor readings.
* Detect patterns that may indicate machine failures.
* Engineer features for predictive maintenance models.
* Build an AI-based system for early failure detection.

## Key Features

| Feature                 | Description                                |
| ----------------------- | ------------------------------------------ |
| UDI                     | Unique identifier for each record          |
| Product ID              | Product identification number              |
| Type                    | Product quality type (L, M, H)             |
| Air Temperature [K]     | Ambient air temperature                    |
| Process Temperature [K] | Process operating temperature              |
| Rotational Speed [rpm]  | Machine rotational speed                   |
| Torque [Nm]             | Torque generated by the machine            |
| Tool Wear [min]         | Tool wear duration in minutes              |
| Machine Failure         | Target variable indicating machine failure |
