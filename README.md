# Predictive-Maintenance-IoT-AI



This project uses the AI4I Predictive Maintenance Dataset.

For detailed information about the dataset, see:
[dataset_overview.md](dataset_overview.md)

## Original Dataset
-Total Rows: 10,000
-Total Columns: 14

## Features
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Machine Failure Prediction
- Model Evaluation

## How to Run
1. Open Jupyter Notebook
2. Install required libraries
3. Run all notebook cells

## Results
The model predicts potential machine failures and helps improve maintenance planning.
